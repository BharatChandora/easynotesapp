About

This fun website has been developed completely from scratch to practice angular basics and demonstrate a complete MEAN stack project.

The project implements various functionality such as login, authentication with JWT, CRUD operations for notes and users, and search which are the most common use cases needed for almost the majority of websites.

Creating this website taught me about how component-based websites work. how to communicate with different components in angular. how frontend talks to the backend through the APIs. how to divde website into components and when to use services.
